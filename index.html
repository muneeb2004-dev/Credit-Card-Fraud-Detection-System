About the project

Core features

The notebook builds a fraud-detection workflow for anonymized credit-card transactions. It covers data exploration, imbalance handling, feature scaling, DNN training, XGBoost training, model comparison, and a final weighted ensemble that produces a fraud probability.

Exploratory analysis Class distribution, amount distribution, correlations, and time patterns.
Imbalance handling Balances 10,000 legitimate and 10,000 fraud examples for model training.
Two-model strategy Compares a neural network with an XGBoost tabular ensemble.
Agent-style decision Combines model probabilities with 40% DNN and 60% XGBoost weighting.
