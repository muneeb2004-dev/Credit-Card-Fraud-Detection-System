Pipeline

Agent workflow

1 Perceive

A transaction arrives from the banking stream with 30 observed features.

2 Process

The DNN learns hidden patterns while XGBoost handles tabular structure.

3 Decide

The agent combines 40% DNN and 60% XGBoost probability scores.

4 Act

A 0.5 threshold classifies the transaction as legitimate or fraud.

About the project

Core features

This project presents a Fraud Detection Agent for real-world financial security. The environment is a banking transaction stream, the observation state is the 30 transaction features, and the policy brain is the combined DNN and XGBoost decision system.

Agentic AI framework Perceive, process, decide, and act on banking transaction observations.
Defined feature state Uses V1-V28 PCA features plus Time and Amount; Class trains the target label.
Deep Neural Network Four-layer 64-32-16-1 architecture with ReLU, Sigmoid, Dropout, and Batch Normalization.
XGBoost ensemble Uses 200 estimators and max depth 6 for strong tabular fraud detection.
Decision layer Combines 40% DNN and 60% XGBoost probabilities with a 0.5 fraud threshold.
Future-ready design Can evolve toward real-time streams, adaptive thresholds, and SHAP explainability.
